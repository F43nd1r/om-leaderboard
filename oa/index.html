Overlap Area Leaderboard

Explanation

The rules of the game would imply that

  • components (inputs, outputs, glyphs, arms) cannot overlap with one another
  • Disposal glyph and Van Berlo's wheel can only appear once in a solution

However, these two restrictions are not enforced by the solution file,
and solutions which violate them can still be run and recorded in game.

Solutions with this behavior can be created using an external editor,
or you can rotate an object, drag it off screen,
shift click another object, and drag it in place,
to achieve overlap within the game itself as shown in this video:

There is no known mechanism to double the disposal glyph or
Van Berlo's wheel within the game,
so this requires an external editor such as F43dit or Fazzone.

When running solutions with this exploit,
it is highly encouraged to enable Hermit Mode in the options,
so as not to overwrite your existing scores.
